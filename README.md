## Usage

Install pipenv first

```
pip install pipenv
```

```
git clone https://github.com/Thebluedragon7/csv2parquet-ngoc.git
```

```
cd csv2parquet-ngoc
```

open `main.py`
and change the following fields to meet your requirements

```
YOUR_CSV_FILE: str = ""
OUTPUT_NAME: str = "output.parquet"
```

```
pipenv install
```

```
pipenv run python main.py
```
